Hello, world!

This is a Jekyll seed application.

Your index page content would show up here.

Stuff this seed app comes with:

  • Font Awesome icons
  • Beautiful, responsive style skeleton based on skeleton.css, fitted out with a lovely collapsing mobile navbar. (Resize the screen to check it out!)
  • Sass files to manage styles
  • Grunt to handle tasks like image resizing, uglifying your JavaScript, and compiling Sass down to CSS and minifying it.
  • Full-site search functionality with lunr.js
  • An install script for Macs to install all the dependencies
  • klipse to include awesome runnable code snippets
  • Categories and tags structure so you can get started adding new ones
